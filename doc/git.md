# git

### 1. **设置个人身份**
```bash
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```
### 2. **初始化**
```bash
git init
```
### 3. **设置远程仓库**
```bash
git remote add <名称> <地址>

```
### 4. **添加**
```bash
git add .
```
### 5. **提交**
```bash
git commit -m "提交描述"
```
### 6. **推到远端**
```bash
git pull
```
